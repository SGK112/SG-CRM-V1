Usage Guidelines

Primary Color (#333333)

Use for main headings, body text, and navigation items. Provides strong readability and professional appearance.

Secondary Color (#B89778)

Use for accents, borders, and secondary elements. Represents warmth and craftsmanship in stone work.

Background Color (#F5F5F5)

Use for main background areas, card backgrounds, and subtle separators. Creates clean, professional look.

Accent Color (#4A90E2)

Use for call-to-action buttons, links, and interactive elements. Draws attention and encourages action.
